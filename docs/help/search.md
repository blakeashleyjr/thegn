# Search

Three search surfaces, one habit: type, arrow, `↵`.

- `Ctrl-Alt-/` — search the **focused pane's** scrollback/history.
- `Ctrl-/` — search **across all panes**.
- The [[command-palette]]'s `/` mode searches **file contents** in the
  focused worktree, and `>` fuzzy-opens files.

Inside this help, `/` searches every page — titles fuzzily, bodies by
substring — and `↵` jumps straight to the matching section.
