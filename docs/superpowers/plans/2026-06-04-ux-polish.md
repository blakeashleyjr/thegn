- [x] **Rewrite toggle logic with state file tracking**

Key design decisions discovered during implementation:

1. **URL format**: `plugin_url()` must use literal `~` (not expanded to `/home/blake/`) to match the exact URL string in KDL layout files. Without this, `zellij pipe` can't find the running plugin and spawns a new floating instance instead.

2. **Show mechanism**: Uses `pipe_plugin(url, "superzej_show")` Ôćĺ plugin calls `show_self(false)`. The `false` argument is critical: `show_self(true)` restores as a floating/centered pane.

3. **No `launch-or-focus-plugin`**: Originally planned as hybrid (pipe for hide, launch-or-focus for show). `launch-or-focus-plugin` creates panes at the current focus position, not the original layout position. Pure pipe-based approach avoids this entirely.
