# Forge Integration Abstraction Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Refactor the existing GitHub integration into a generic `Forge` trait to support multiple forges (GitHub, GitLab, Forgejo, etc.) while retaining all current capabilities (Issues, PRs, check status).

**Architecture:** We will introduce a `Forge` trait in `crates/superzej-core/src/forge/mod.rs` that defines the contract for forge operations. The existing GitHub implementation in `crates/superzej-core/src/github.rs` will be adapted to implement this trait. We will also add a `Forgejo` implementation as a proof of concept for the abstraction. The `PanelState` and `PrStatus` models will be slightly generalized if necessary, but mostly kept intact as they represent generic PR/MR concepts. The CLI will be updated to select the correct forge implementation based on the remote URL.

**Tech Stack:** Rust, `serde`, `serde_json`, shell execution (for `gh`, `glab`, `tea` or direct REST API via `reqwest` if preferred, but we'll stick to shelling out for CLI tools to match existing architecture if possible, or use `reqwest` for generic REST if CLIs are missing). *Assumption for this plan: We will use CLI wrappers for now, similar to the existing `gh` wrapper, to avoid adding `reqwest` as a heavy dependency unless necessary. We will define the trait and refactor GitHub.*

```rust
use crate::forge::models::*;
use crate::forge::Forge;
use crate::remote::GitLoc;

pub struct ForgejoForge;

fn tea_out(loc: &GitLoc, args: &[&str]) -> Result<String, ForgeError> {
    let out = loc.command("tea").args(args).output().map_err(|e| ForgeError::Other(e.to_string()))?;
    if out.status.success() {
        return Ok(String::from_utf8_lossy(&out.stdout).trim().to_string());
    }
    Err(classify(&String::from_utf8_lossy(&out.stderr).to_lowercase()))
}

fn tea_run(loc: &GitLoc, args: &[&str]) -> Result<(), ForgeError> {
    let out = loc.command("tea").args(args).output().map_err(|e| ForgeError::Other(e.to_string()))?;
    if out.status.success() {
        Ok(())
    } else {
        Err(classify(&String::from_utf8_lossy(&out.stderr).to_lowercase()))
    }
}

fn classify(stderr: &str) -> ForgeError {
    if stderr.contains("command not found") || stderr.contains("no such file") {
        ForgeError::NotInstalled
    } else if stderr.contains("not logged in") || stderr.contains("unauthorized") {
        ForgeError::NotAuthenticated
    } else if stderr.contains("no pull request") {
        ForgeError::NoPr
    } else {
        ForgeError::Other(stderr.trim().to_string())
    }
}

impl Forge for ForgejoForge {
    fn pr_status(&self, loc: &GitLoc) -> PrPanel {
        let branch = loc.git_out(&["rev-parse", "--abbrev-ref", "HEAD"]).unwrap_or_default();
        // Map `tea pr list` or similar to PrStatus. This requires mapping tea's output.
        // For POC, return an error indicating WIP if not fully mapped, or implement a basic mapping.
        PrPanel {
            state: PanelState::Error { message: "Forgejo PR status mapping WIP".into() },
            worktree: loc.path(),
            branch,
            fetched_at: crate::util::now(),
        }
    }
    fn create_pr(&self, loc: &GitLoc, opts: &CreateOpts) -> Result<String, ForgeError> {
        // Map opts to `tea pr create` args
        tea_out(loc, &["pr", "create"])
    }
    fn open_pr(&self, loc: &GitLoc) -> Result<(), ForgeError> {
        tea_run(loc, &["pr", "open"])
    }
    fn approve_pr(&self, _loc: &GitLoc, _body: Option<&str>) -> Result<(), ForgeError> {
        Err(ForgeError::Other("Approve not yet mapped for tea".into()))
    }
    fn merge_pr(&self, loc: &GitLoc, _method: MergeMethod, _delete_branch: bool, _auto: bool) -> Result<(), ForgeError> {
         tea_run(loc, &["pr", "merge"])
    }
    fn reviews(&self, _loc: &GitLoc) -> Result<String, ForgeError> {
        Err(ForgeError::Other("Reviews not yet mapped for tea".into()))
    }
    fn rerun_failed_checks(&self, _loc: &GitLoc) -> Result<u32, ForgeError> {
        Err(ForgeError::Other("Checks rerun not yet mapped for tea".into()))
    }
}
```
