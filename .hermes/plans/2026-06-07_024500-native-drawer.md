# Native Files Drawer Implementation Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Implement the native bottom drawer overlay for the file manager (Yazi), dropping the old WASM/zellij dependency.
**Architecture:** The native `thegn` compositor will manage a `drawer: Option<PtyPane>` directly inside `run.rs`. When active, the drawer renders overlaying the bottom rows of the screen (typically `drawer.height`).
**Tech Stack:** Rust, `thegn` native `PtyPane`, `thegn-core::yazi`.

---

**Step 2: Run test to verify failure**

**Step 3: Write minimal implementation**
