# Keybind System Design for superzej (Native Host)

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Design and implement a full keybind system for `szhost` (superzej's native host) supporting vim/emacs modes and custom bindings, moving past Zellij limitations.

**Architecture:**

- Expand the current `termwiz` KeyCode/Modifiers handling into an abstract `Chord` matching system supporting multiple modes (Normal, Vim Normal, Vim Insert, Emacs).
- Use `taffy` or the current `CenterTree` abstraction to dispatch actions conditionally based on focus and mode state.
- Keep backwards compatibility with the current `config.toml` `[keybinds]` and `[[actions]]` syntax.
- Extend `Action` enum to include mode-switching verbs.
- Abstract the input matching logic to use a state machine (handling leader keys / sequences like `g g`).

**Tech Stack:** `superzej-host`, `superzej-core`, `termwiz`

// in crates/superzej-host/src/sequence.rs
use termwiz::input::{KeyCode, Modifiers};
use crate::keymap::Action;

#[derive(Debug, Clone, PartialEq, Eq)]
pub struct Key {
    pub code: KeyCode,
    pub mods: Modifiers,
}

#[derive(Debug, PartialEq, Eq)]
pub enum MatchResult {
    None,
    Pending,
    Matched(Action),
}

pub struct SequenceMatcher {
    sequences: Vec<(Vec<Key>, Action)>,
    current: Vec<Key>,
}

impl SequenceMatcher {
    pub fn new() -> Self {
        Self { sequences: Vec::new(), current: Vec::new() }
    }

    pub fn add_sequence(&mut self, seq: Vec<Key>, action: Action) {
        self.sequences.push((seq, action));
    }

    pub fn feed(&mut self, key: Key) -> MatchResult {
        self.current.push(key);

        let mut possible = false;
        for (seq, action) in &self.sequences {
            if seq.starts_with(&self.current) {
                if seq.len() == self.current.len() {
                    self.current.clear();
                    return MatchResult::Matched(*action);
                }
                possible = true;
            }
        }

        if possible {
            MatchResult::Pending
        } else {
            self.current.clear();
            MatchResult::None
        }
    }
}
```

// in crates/superzej-host/src/keymap.rs
use std::collections::HashMap;
use crate::sequence::{Key, SequenceMatcher, MatchResult};

pub struct KeyMap {
    modes: HashMap<Mode, SequenceMatcher>,
}

impl KeyMap {
    pub fn new() -> Self {
        Self { modes: HashMap::new() }
    }

    pub fn insert(&mut self, mode: Mode, seq: Vec<Key>, action: Action) {
        self.modes.entry(mode).or_insert_else(SequenceMatcher::new).add_sequence(seq, action);
    }

    pub fn dispatch(&mut self, mode: Mode, key: Key) -> MatchResult {
        if let Some(matcher) = self.modes.get_mut(&mode) {
            matcher.feed(key)
        } else {
            MatchResult::None
        }
    }
}
```
