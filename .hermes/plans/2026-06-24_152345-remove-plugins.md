# Superzej Plugin/Dashboard/Chat Removal Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Remove the dashboard app tile, the chat tile, the `apps/` submodules, and the plugin-related APIs from `sz-kit` and the compositor, making the shell lighter and pure workspace-centric.

**Architecture:** We are dropping the dashboard and chat features, eliminating the submodules (`termite-chat`, `termite-agent` etc.), removing `superzej-dashboard` from the workspace, and deleting the top-level app tabs. The proxy crate's chat endpoints (which simulate the OpenAI chat completions API) will remain untouched as they serve external integrations (the proxy is independent of the removed UI tiles).

**Tech Stack:** Rust, Cargo
