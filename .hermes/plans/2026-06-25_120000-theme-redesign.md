# Theme System Redesign & Migration Plan

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Provide 20 high-quality, meticulously designed themes (replacing the current low-quality generic ones while retaining primary defaults) and audit/improve the UX/UI of theme switching across the application.

**Architecture:** The theming engine resides in `crates/thegn-core/src/theme.rs` utilizing an enum-backed preset list that builds `Palette` structures via 12-slot `R;G;B` arrays + derived extensions. The selection logic is exposed via `Config`, while UI/UX switching may require changes to the picker implementations (handled in `crates/thegn-host`).

**Tech Stack:** Rust (thegn crates), Custom config enum macro for dynamic selection, Terminal SGR sequences (`R;G;B`).
